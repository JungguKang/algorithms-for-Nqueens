# algorithms-for-Nqueens
This repository contains several algorithms to solve N-queens algorithm.
## N-Queens problem
![nqueens](https://user-images.githubusercontent.com/57992058/87848163-748cbd00-c918-11ea-9b3c-f0680816c150.PNG)   
There is an chess board which is a size of NxN, and must allocate N numbers of queens.   
Each queen can attack all horizontal, vertical and diagonal blocks from her standing block.   
In this N-Queens problem, each Queen must be allocated not attacking each other.
## Algorithms used for this problem
> 1. Uninformed search
>    * DFS
>    * BFS
>    * DFID
> 2. Hill Climbing search
> 3. Genetic algorithm
> 4. SAT solver
